## API 设计

### 论文管理 API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/papers` | 获取论文列表 |
| POST | `/api/papers` | 添加论文（传入 arXiv ID/URL） |
| GET | `/api/papers/{id}` | 获取单篇论文详情 |
| DELETE | `/api/papers/{id}` | 删除论文 |
| GET | `/api/papers/{id}/pdf` | 获取 PDF 文件 |
| GET | `/api/papers/{id}/export` | 导出 PDF |

### 对话 API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/chat/{paper_id}` | 发送消息（流式返回） |
| GET | `/api/chat/{paper_id}/history` | 获取对话历史 |
| DELETE | `/api/chat/{paper_id}/history` | 清空对话历史 |

### 请求/响应示例

**添加论文**
```json
// POST /api/papers
// Request
{
  "arxiv_input": "2301.12345"  // 或完整 URL
}

// Response
{
  "id": "2301.12345",
  "title": "Attention Is All You Need",
  "authors": ["Ashish Vaswani", "..."],
  "summary": "We propose a new simple network architecture...",
  "download_time": "2024-01-15T10:30:00Z",
  "pdf_path": "~/.ipaper/papers/2301.12345/paper.pdf"
}
```

**发送对话消息**
```json
// POST /api/chat/2301.12345
// Request
{
  "message": "这篇论文的主要贡献是什么？",
  "selected_text": null,  // 可选：用户选中的文本
  "context": {
    "page": 3,
    "section": "Introduction"
  }
}

// Response (SSE 流式)
data: {"type": "chunk", "content": "这篇"}
data: {"type": "chunk", "content": "论文"}
data: {"type": "chunk", "content": "的主要贡献..."}
data: {"type": "done", "full_response": "这篇论文的主要贡献..."}
```

## LLM 对话设计

### 对话上下文构建

```
┌─────────────────────────────────────────────────────────────┐
│                    Prompt 结构                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  System Prompt:                                             │
│  "你是一个学术论文阅读助手，帮助用户理解论文内容..."           │
│                                                             │
│  ─────────────────────────────────────────────────────────  │
│                                                             │
│  论文 PDF（完整发送给 Gemini，利用其多模态能力）              │
│                                                             │
│  ─────────────────────────────────────────────────────────  │
│                                                             │
│  对话历史:                                                   │
│  User: 这篇论文讲了什么？                                    │
│  Assistant: 这篇论文提出了...                                │
│  User: {当前问题}                                           │
│                                                             │
│  ─────────────────────────────────────────────────────────  │
│                                                             │
│  [可选] 用户选中的文本:                                      │
│  "用户选中了以下内容，请针对这部分回答：{selected_text}"      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### System Prompt 设计

```
你是一个专业的学术论文阅读助手。你的任务是帮助用户理解论文内容。

职责：
1. 解释论文中的概念、方法和结论
2. 回答用户关于论文的问题
3. 如果用户选中了特定文本，优先针对该内容回答
4. 使用中文回答，但保留专业术语的英文原文（如 "Transformer"）

注意事项：
1. 基于论文内容回答，不要编造论文中没有的信息
2. 如果不确定，请明确说明
3. 可以引用论文中的具体章节或图表
```

### 流式响应处理

```python
# 后端 (FastAPI)
@router.post("/chat/{paper_id}")
async def chat(paper_id: str, request: ChatRequest):
    async def generate():
        async for chunk in llm_service.chat_stream(
            paper_id=paper_id,
            message=request.message,
            selected_text=request.selected_text
        ):
            yield f"data: {json.dumps({'type': 'chunk', 'content': chunk})}\n\n"
        
        yield f"data: {json.dumps({'type': 'done'})}\n\n"
    
    return StreamingResponse(generate(), media_type="text/event-stream")
```
